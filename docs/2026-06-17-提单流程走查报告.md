## 四、本轮新增:左侧菜单 + 退出登录
- `/ops` 左侧菜单补齐**原版全部菜单**:运营总览 / 项目池 / 人员进度 / 需求提单 / 环节交付配置 / 管理员

  - **运营总览 / 项目池 / 人员进度 / 管理员** → 旧应用 `/?view=key`

- 注:暂**未按角色隐藏「管理员」**(旧应用本身会对非管理员收起,深链过去会回落到其可见页);若要在 `/ops` 菜单层面按角色控制,可再加一次 `/me` 拉取
