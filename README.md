# companyPlan

Playable ad production SaaS prototype for demand-ticket, project, people, and admin visibility workflows.

## Run

```bash
npm install
npm run dev -- --port 5174
```

## Build

```bash
npm run build
```

## Scope

- Frontend happy-path prototype.
- KDocs/WPS-style demand-ticket table.
- Role-scoped views for admin and non-admin users.
- Ticket attachments, status grouping, worksheet tabs, and admin/global panels.

The Codex skill for this project is stored in `skills/company-plan` and installed locally at `~/.codex/skills/company-plan`.
